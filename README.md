### 🔐 TP-Crypto
## 📖 Overview

TP-Crypto is an educational cryptography project that implements and demonstrates fundamental and modern cryptographic algorithms.

It is designed to help students understand how cryptographic systems work internally through practical implementations rather than only theory.

The project covers:

1. Classical Cryptography
2.Symmetric Encryption
3. Asymmetric Encryption
4. Key Exchange Protocols
5. Hash Functions
6. Digital Signatures
## Objectives

This project aims to:

Understand how encryption and decryption algorithms work
Compare symmetric vs asymmetric cryptography
Learn the role of hash functions in data integrity
Explore digital signatures and authentication systems
Build a strong foundation in cybersecurity principles
